Git - распределенная система контроля версий, записывающая изменения в файл или набор файлов в течение времени и позволяющая вернуться позже к определенной версии

У Git есть три основных состояния, в которых могут находиться файлы:
1. Зафиксированное (committed) - файл уже сохранен в вашей локальной базе 
2. Измененное (modified) - файлы, которые поменялись, но еще не были зафиксированы
3. Подготовленное (staged) - измененные файлы, отмеченные для включения в следующий коммит

Отличия Git и GitHub
Git - система контроля версий 
GitHub - это онлайн сервис, который предоставляет хранилище
